# La Verrière Gourmande

Vitrine de produits de **La Verrière Gourmande** permettant aux clients de parcourir l'offre, créer un panier temporaire, et passer commande par téléphone. Idéal pour ceux qui ne peuvent pas se présenter aux évènements.

## Fonctionnalités

- Catalogue des produits
- Panier temporaire
- Contact par téléphone pour finaliser la commande

## À propos

La Verrière Gourmande propose une sélection de produits locaux. Visitez [laverrieregourmande.com](https://laverrieregourmande.com/) pour plus d'informations.
